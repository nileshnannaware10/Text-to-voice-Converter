# Text-to-Voice Converter

A simple web application that converts text input into speech, developed using **HTML**, **CSS**, and **JavaScript**. This project showcases how to leverage the Web Speech API to create a text-to-voice functionality.

## Features

- Input any text and listen to it as speech.
- Customizable voice options (if supported by the browser).
- Simple and user-friendly interface.
- Fully responsive design for optimal use on different devices.

## Technologies Used

- **HTML**: Structure of the application.
- **CSS**: Styling for a visually appealing and responsive UI.
- **JavaScript**: Core functionality for text-to-speech conversion.
